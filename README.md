# Examining the Effects of Clinicopathologic Parameters on Time to Relapse in Primary Breast Cancer Patients: A Multiple Linear Regression Analysis
## Authors: Kaleb Chaney and Reece Champion 
This research project was conducted in December 2025 and served as the capstone project for SDS 324E: Elements of Regression Analysis taught by Dr. Antonio Linero at The University of Texas at Austin. The study analyzes various clinicopathologic parameters and how they relate to a breast cancer patient's time to relapse.
